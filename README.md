# Enterprise Banking Fraud Analytics System

## Project Overview
This project is an end-to-end Banking Fraud Detection and Analytics System built using SQL Server, SSIS, and Tableau.

The system performs:
- ETL processing
- Data cleaning and transformation
- Fraud detection analysis
- Data warehousing
- Interactive business intelligence visualization

The project simulates enterprise-level banking fraud monitoring workflows.

---

# Technologies Used

- SQL Server
- SQL Server Integration Services (SSIS)
- Tableau
- SQL
- CSV Datasets

---

# Project Architecture

Raw CSV Data
→ SSIS ETL Pipeline
→ Staging Tables
→ Cleaned Tables
→ Fraud Detection Logic
→ Data Warehouse
→ Tableau Dashboard
